# 视频大纲生成

快速生成2-3个视频脚本大纲方案，包含标题、封面建议、结构设计。

## 何时使用

- 保存视频brief后
- 用户说"提供大纲"、"给几个方案"
- 视频创作流程的Step 3（大纲讨论）

## 核心功能

1. **提供2-3个大纲方案**：每个方案不同角度
2. **每个方案包含**：标题、封面建议、核心角度、大纲结构、时长预估
3. **应用封标策略**：5种强对比公式、封面设计策略

## 大纲方案模板

```markdown
## 方案1：[吸引人的标题]
**封面建议**：结果展示/人脸（根据粉丝量）
**核心角度**：xxx
**时长预估**：⏱️ 5-8分钟

### 大纲结构
1. 开头（30秒-1分钟）：确认标题承诺 + 建立悬念
2. 核心模块1（2分钟）：xxx
3. 核心模块2（2分钟）：xxx
4. 意外惊喜（1分钟）：超出预期的内容
5. 结尾（1-2分钟）：兑现承诺 + Call to Action

### 需要演示的内容
- 演示1：xxx
- 演示2：xxx

### 优势劣势
✅ 优势：xxx
⚠️ 劣势：xxx
```

## 标题策略（5种对比）

1. 数量对比："1个功能 vs 10个工具"
2. 价格对比："免费 vs 付费，差距有多大？"
3. 结果对比："0粉到30万粉，我做对了什么"
4. 强弱对比："新手 vs Cursor，谁能赢？"
5. 时间对比："3分钟 vs 3小时"

## 封面策略

- **< 10万粉**：结果展示封面优先
- **> 10万粉**：可测试人脸封面
- **教程类**：Before/After对比
- **评测类**：产品特写+数据

## 注意事项

- 应用内容承接策略（开头确认→中段惊喜→结尾兑现）
- 时长预估要合理
- 等待用户选择，不假设
